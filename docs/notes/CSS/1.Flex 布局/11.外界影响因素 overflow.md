## 一、简介

`overflow: hidden` 是 CSS 中用于控制元素内容溢出行为的属性。当一个元素的内容超出其指定的宽度和高度时，`overflow: hidden` 会将溢出的部分隐藏，而不是显示滚动条或让内容溢出到容器外部。

给第二个元素添加 `overflow: hidden`，效果类似 `min-width`，但是多余的部分被遮挡了
