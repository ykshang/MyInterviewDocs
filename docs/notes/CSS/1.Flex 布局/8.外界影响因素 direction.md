## 一、简介

direction 是 CSS 中的一个属性，用于设置文本、表格列和水平溢出的方向。它主要控制文本的书写方向以及元素的排列方向。

```css
direction: ltr | rtl | inherit;
```

- **`ltr`**（默认值）：从左到右（Left-to-Right），适用于大多数语言，如英语、中文等。
- **`rtl`**：从右到左（Right-to-Left），适用于希伯来语、阿拉伯语等从右向左书写的语言。
- **`inherit`**：继承父元素的方向设置。
