## 一、简介

`margin` 是 CSS 中用于控制元素外边距的属性。外边距是元素边框与周围元素之间的空间。通过调整 `margin`，可以控制元素之间的间距以及元素在页面中的布局。

### 1. 语法
```css
margin: value;
```
### 2. 取值
**`value`** 可以是以下值：

- 长度值（如 `10px`、`2em`）。
- 百分比（如 `5%`）。
- `auto`：浏览器自动计算外边距（常用于水平居中）。
- `inherit`：继承父元素的外边距。
